Class DatasetShell

A shell to provide a unified way of handling the various ways a graph can be represented: a full blown RDF Dataset core instance, or an Array of Quads, or a Set of Quads.

Remarks

The reason this is necessary is (1) the Array object in JS does not have a add property and (2) care should be taken about creating new RDF Datasets, see the new method.
